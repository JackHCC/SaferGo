## 设计
- 针对滴滴出行安全
- 2018年，针对滴滴安全问题，提出**实时定位守护**的概念
- 具体详细说明参考官网说明[SaferGo-Web](https://blog.creativecc.cn/Web-SaferGo/)

## 版本迭代
iSafe(2018) -> SaferGo(2019)

## 荣誉与说明
- iSafe:2018华北五省暨港澳台大学生计算机应用大赛本科组一等奖
- SaferGo:2019计算机设计大赛国赛一等奖
